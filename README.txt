== Description ==

This plugin adds a new block and a shortcode for WooCommerce product cross-sells. You will be able to use them to display product cross-sells on product pages.
This plugin also let's you to choose hide related products on product pages.

If you need help with any customizations, please send me an email to dhanuka@hashcodeab.se

== Dependancies ==

1. WooCommerce
2. Advanced Custom Fields
